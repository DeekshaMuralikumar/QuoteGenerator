A minimalist React app that displays curated inspiration in a "Museum Gallery" style.

---

### **## Setup**

1. **Install**: `npm install`
2. **Run**: `npm start`

---

### **## Core Components**

* **App.js**: Manages state and fetches random quotes from the **DummyJSON API**.
* **QuoteCard.js**: A compact, beige card displaying "Daily Wisdom".
* **App.css**: Handles the centered Flexbox layout and paper-like aesthetic.

---

### **## Features**

* **API Integration**: Uses `https://dummyjson.com/quotes/random` for reliable data.
* **Minimalist UI**: Features a beige background and serif typography.
* **Centered Layout**: The card and "New Quote" button to get a new quote.
